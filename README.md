# ICFS-tools
InterCLASS Filtering Serviceの補助ツール群です。ログファイルのCSVの簡易解析機能など、便利に使っていただけるツール群をオープンソースで提供いたします。ソースコードについては極力PowerShellで提供いたします。データの処理量が多い場合はPython3にて書くこともございます。予めご了承ください。
